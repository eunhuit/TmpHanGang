## 프로젝트 소개
한강물 온도를 파이썬으로 불러와 보자 추가로 저장된 명언도 같이 불러와 보자!

<br>

## 구현 기능

### 기능 1
      openpyxl 라이브러리로 명언 데이터를 액셀화 시킴
### 기능 2
      API를 활용해 정확한 물 온도
### 기능 3
      가슴 따뜻해지는 명언으로 인생의 생기를 회복!
